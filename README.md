# ShikshaSetu-frontend

### Setup instruction 

1. Clone the project
```
    git clone https://github.com/gauravaundkar017/ShikshaSetu-frontend.git
```
2. Move into the directory
```
    cd ShikshaSetu-frontend
```
3. Install dependencies 
```
    npm i
```

4. Run the server 
```
    npm run dev
```
